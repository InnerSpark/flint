# Flint brand

The Flint mark is a faceted flint shard, struck to throw a spark (a nod to the parent brand, Inner Spark). The five facets echo the layered system (primitives, semantic, components), and the mark is drawn entirely from Flint's own blue ramp.

## Files

- `flint-logo.svg` — primary lockup (mark + wordmark), for light backgrounds
- `flint-logo-lowercase.svg` — lowercase wordmark variant
- `flint-icon.svg` — mark only, for light backgrounds (app icon, avatar)
- `flint-icon-inverse.svg` — mark for dark backgrounds (lighter facets)
- `flint-icon-mono.svg` — single-ink tonal mark; uses `currentColor`, so set `color` to ink it
- `favicon.svg` — the mark as an SVG favicon

## Usage

- Clear space: keep at least the width of the spark (about the mark height divided by 4) clear on all sides.
- Minimum size: 16px for the icon, 88px wide for the lockup.
- Color: the blue-ramp mark on light backgrounds, the inverse (lighter facets) on dark. The mono version inherits `currentColor`, so it inks to any single color.
- Don't: recolor the facets off the blue ramp, add effects (shadows, gradients, glow), stretch, rotate, or place the light mark on a dark background (use the inverse).

## Wordmark

The lockups use live Open Sans ExtraBold text. For print, or anywhere the font may be missing, outline the wordmark to paths so it renders identically everywhere.
